## Description

O "Batepapo" é um aplicativo web inspirado em sites de conversas online, onde por meio das tecnologias de back-end, possue as mesma funções de chat como aplicativos conhecidos, por exemplo o Bate Papo Uol


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
